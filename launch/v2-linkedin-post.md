## Post

Introducing TokenTool MCP — the first MCP server for compliant token issuance.

AI agents can now deploy, manage, and query tokenized assets directly through Bitbond's TokenTool infrastructure. No Solidity required.

What the TokenTool MCP server enables:

→ Deploy CertiK-audited tokens on 10 networks (Ethereum, Polygon, Arbitrum, Base, Solana, Stellar, and more)
→ Built-in compliance: whitelist, blacklist, pausable, force transfer, document URI
→ Full lifecycle management: mint, burn, pause, transfer, query — all via natural language
→ Works with Claude Desktop, Cursor, VS Code, and any MCP-compatible AI client

TokenTool has powered 8,300+ real-world token deployments since 2020. The MCP server wraps the same production infrastructure — same CertiK-audited contracts, same reliability.

Model Context Protocol (MCP) is the open standard that connects AI agents to real-world tools and data. Stripe, Alpaca, and Griffin have already adopted it for payments, trading, and banking. TokenTool MCP brings tokenized securities into the agentic finance stack.

Open source. MIT licensed. Free on testnet.

GitHub, npm install instructions, and documentation in the comments below.
