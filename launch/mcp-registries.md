# MCP Registries — Where to Submit TokenTool MCP

## Tier 1 — Must Submit (highest impact)

### 1. Official MCP Registry
- **URL**: https://registry.modelcontextprotocol.io / https://github.com/modelcontextprotocol/registry
- **Submit**: PR to GitHub repo (must be published to npm first)
- **Reach**: THE canonical source — all MCP clients will pull from this
- **Requirements**: npm/PyPI published package, metadata JSON
- **Priority**: ⭐⭐⭐⭐⭐

### 2. Awesome MCP Servers (GitHub)
- **URL**: https://github.com/punkpeye/awesome-mcp-servers
- **Submit**: PR adding entry to README
- **Reach**: 35K+ stars, most-referenced MCP list
- **Requirements**: Working MCP server, good README
- **Priority**: ⭐⭐⭐⭐⭐

### 3. MCP.so
- **URL**: https://mcp.so
- **Submit**: Create GitHub issue in their repo
- **Reach**: 16,670+ servers listed, major discovery hub
- **Requirements**: GitHub repo link, description
- **Priority**: ⭐⭐⭐⭐⭐

### 4. Smithery.ai
- **URL**: https://smithery.ai
- **Submit**: npm publish + register via their CLI/dashboard
- **Reach**: Popular MCP marketplace, install metrics
- **Requirements**: npm package, smithery.yaml config
- **Priority**: ⭐⭐⭐⭐

## Tier 2 — Should Submit (good reach)

### 5. Glama.ai
- **URL**: https://glama.ai/mcp/servers
- **Submit**: Submit via website form
- **Reach**: Curated directory with categories
- **Requirements**: Working server, description
- **Priority**: ⭐⭐⭐⭐

### 6. PulseMCP
- **URL**: https://pulsemcp.com/servers
- **Submit**: Auto-indexes from npm/GitHub, can also submit manually
- **Reach**: 8,610+ servers, daily updated
- **Requirements**: Published package
- **Priority**: ⭐⭐⭐⭐

### 7. ClawHub
- **URL**: https://clawhub.com
- **Submit**: `clawhub publish` CLI
- **Reach**: OpenClaw ecosystem specifically
- **Requirements**: Skill folder structure
- **Priority**: ⭐⭐⭐

### 8. GitHub MCP Directory
- **URL**: https://github.com/mcp
- **Submit**: PR / discussion in modelcontextprotocol org
- **Reach**: GitHub-native, developer trust signals
- **Requirements**: GitHub repo
- **Priority**: ⭐⭐⭐

## Tier 3 — Nice to Have

### 9. npm (package registry)
- **URL**: https://npmjs.com
- **Submit**: `npm publish`
- **Reach**: Massive — all Node.js devs search here
- **Requirements**: npm account, package.json ready ✅
- **Note**: This is prerequisite for most other registries

### 10. Dev.to / Medium / Hashnode (content)
- **Submit**: Publish tutorial/announcement post
- **Reach**: Developer blogs, SEO long-tail
- **Note**: Not a registry but drives discovery

## Submission Order
1. npm publish (prerequisite)
2. Official MCP Registry (PR)
3. Awesome MCP Servers (PR)
4. MCP.so (GitHub issue)
5. Smithery.ai (register)
6. Glama.ai (form)
7. PulseMCP (auto-indexes from npm)
8. ClawHub (publish)
9. Blog posts (Dev.to, Medium)
